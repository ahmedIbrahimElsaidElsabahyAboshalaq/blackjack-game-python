# 🃏 Blackjack Game

A console-based Blackjack game built with Python.

## Features

- Draw random cards
- Hit or Stand
- Dealer follows Blackjack rules
- Ace automatically counts as 1 or 11
- Play multiple rounds

## Requirements

- Python 3.x

## How to Run

```bash
python main.py
```

## Project Structure

```
blackjack-game-python/
├── main.py
└── art.py
```
